## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/sumit-616/shopping-app.git
    cd shopping-app
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Start the development server:

    ```bash
    npm run dev
    ```

4. Open the app in your browser:

    ```
    http://localhost:5173
    ```
